## Contents

### 01_Introduction_to_DBMS.md
Introduction to DBMS, advantages over file systems, database users, DBMS architecture, and basic concepts.

### 02_ER_Model.md
Entity-Relationship (ER) Model including entities, attributes, relationships, cardinality, ER diagram symbols, and database design fundamentals.

### 03_Relational_Model.md
Relational model concepts, relations, tuples, attributes, domains, schemas, and relational database structure.

### 04_Keys_in_DBMS.md
Different types of keys such as Primary Key, Candidate Key, Super Key, Alternate Key, Composite Key, and Foreign Key with examples.

### 05_Normalization.md
Database normalization, functional dependencies, anomalies, and normal forms (1NF, 2NF, 3NF, BCNF).

### 06_SQL_Basics.md
SQL fundamentals including DDL, DML, DCL, TCL commands, CRUD operations, and common SQL queries.

### 07_Joins.md
SQL joins including INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN, SELF JOIN, and CROSS JOIN with examples.

### 08_Transactions.md
Database transactions, ACID properties, transaction states, commit, rollback, and recovery concepts.

### 09_Concurrency_Control.md
Concurrency issues, locking protocols, deadlocks, serializability, and transaction scheduling.

### 10_Indexing.md
Indexing concepts, types of indexes, B-Tree indexing, hashing, and performance optimization.
